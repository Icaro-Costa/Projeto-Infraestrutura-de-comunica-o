# Projeto (Semana 1): Planejamento do Chat Simples

## 1. Documento de Planejamento Inicial

### Tema
Implementação de um chat cliente-servidor simples via terminal.

### Escopo
O objetivo final do projeto (nas próximas 4 semanas) é permitir que dois usuários troquem mensagens de texto em tempo real através de uma rede local.

### Tecnologias
* **Linguagem:** Python 3
* **Bibliotecas Planejadas:**
    * `socket`: (Marco da Semana 1) Será usada para a funcionalidade principal de rede (conexão TCP).
    * `threading`: (Planejada para Semana 2) Será necessária para permitir o envio e recebimento de mensagens ao mesmo tempo.
